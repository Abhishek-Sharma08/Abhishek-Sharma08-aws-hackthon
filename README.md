Snowvault is an interactive, AI-powered platform that makes learning to code as engaging as playing a game. Inspired by Duolingo, it offers bite-sized lessons, an interactive coding environment, and a friendly AI companion to guide users through their programming journey. 

## ‚ú® Features

* **Bite-Sized Lessons:** Curriculum broken down into quick, easily digestible modules with smooth page transitions powered by Framer Motion.
* **Gemini-Powered AI Tutor:** Your personal guide through Snowvault! Powered by the Google Gemini API, the tutor provides intelligent hints, encouragement, and real-time feedback on your code.
* **Pro-Level Code Execution:** Practice what you learn instantly. The app features an integrated Monaco Editor (the core of VS Code) and executes code securely via the Piston API.
* **Gamified Learning:** Earn XP, maintain daily levels and streaks, and conquer missions as you progress. Complex app states are seamlessly managed by Redux Toolkit.
* **Custom Authentication:** Secure user authentication flow using JWT and Bcrypt to securely handle user progress and sessions.
* **Sleek UI/UX:** A clean, dark-themed, and intuitive interface designed with Tailwind CSS and Lucide React icons to keep learners focused and motivated.

## üõ†Ô∏è Tech Stack

**Frontend (Vite + React 19)**
* **State Management:** Redux Toolkit
* **Routing:** React Router DOM
* **Styling & UI:** Tailwind CSS v4, Framer Motion, Lucide React
* **Code Editor:** `@monaco-editor/react`
* **HTTP Client:** Axios

**Backend (Node.js + Express 5)**
* **Database:** MongoDB with Mongoose
* **Authentication:** JSON Web Tokens (JWT) & Bcryptjs
* **External APIs:** Google Gemini API (AI Tutor & Feedback), Piston API (Remote Code Execution)
